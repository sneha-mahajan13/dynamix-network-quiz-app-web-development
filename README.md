# Quiz App 

As part of my **Web Development Internship at Dynamix Networks**, I created this web-based **Quiz Application** utilizing **HTML, CSS, and JavaScript**.
Users can track their progress, complete quiz questions from several categories, and receive their final result along with feedback using this program.
# Attributes
Start the quiz screen with a greeting.
One question at a time; a progress bar to indicate quiz completion; a category selection (Web Development, Math, General Knowledge); randomly generated questions and choices on each quiz attempt;
The final score and feedback screen
The opportunity to restart the test
# Tech Stack
JavaScript, HTML, and CSS
